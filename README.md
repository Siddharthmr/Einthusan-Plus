## Installation

1. **Clone the repo**

    ```bash
    git clone https://github.com/Siddharthmr/Einthusan-Plus.git
    ```

2. **Load the extension**

    - Open `chrome://extensions`
    - Enable **Developer mode**
    - Click **Load unpacked** and select the `extension/` folder
